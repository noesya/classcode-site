# Osuny hugo template

Pour installer Hugo
```
brew install hugo
```
Mise à jour avec
```
brew upgrade hugo
```


Pour cloner avec le thème
```
git clone git@github.com:noesya/osuny-hugo-template.git --recurse-submodules
```
Pour récupérer le thème
```
git pull --recurse-submodules
```


Pour lancer le site
```
yarn
hugo server
```
